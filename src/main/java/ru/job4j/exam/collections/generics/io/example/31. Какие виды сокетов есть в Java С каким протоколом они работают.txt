31. Какие виды сокетов есть в Java С каким протоколом они работают

Java для работы в сети имеет специальный пакет java.net, мы будем использовать два класса из этого пакета: 

java.net.ServerSocket - класс реализует серверный сокет, который ожидает запросы, приходящие от клиентов по сети, и может отправлять ответ. 
java.net.Socket - класс реализует клиентский сокет. 
Соединение между java - сокетами устанавливается с помощью транспортного протокола TCP. То есть используя сокеты мы можем передавать и получать информацию по установленному TCP - соединению. 
Сокеты по протоколу TCP/IP служат для реализации надежных двунаправ­
ленных, постоянных, двухточечных, потоковых соединений между хостами
в Интернете. Сокет может служить для подключения системы ввода-вывода в Java
к другим программам, которые могут находиться как на локальной машине, так
и на любой другой машине в Интернете.
В языке Java поддерживаются две разновидности сокетов по протоколу TCP/IP:
один -
для серверов, другой -
для клиентов. Класс ServerSocket служит "при­
емником'; ожидая подключения клиентов прежде, чем предпринять какие-нибудь
действия. Иными словами, класс ServerSocket предназначен для серверов, тогда
как класс Socket -
для клиентов. Он служит для подключения к серверным соке­

чаще всего применяются в прикладных программах на Java, поэтому они здесь и рас­
сматриваются.
