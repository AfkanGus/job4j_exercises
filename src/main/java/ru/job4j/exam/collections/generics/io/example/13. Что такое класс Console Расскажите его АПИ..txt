13. Что такое класс Console Расскажите его АПИ.

Класс Console предназначен для работы с консолью - командной строкой Windows или терминалом Linux/MacOS и упрощает работу с ними. Класс Console определён в пакете java.io. 
Класс Console служит для ввода-вывода данных на консоль, если таковая име­
ется, и реализует интерфейс FlushaЫe.поддерживает операцию сброса буферизованных данных.
Конструкторы в классе Console не предоставляются.

Когда вы запускаете программу Java из командной строки операционной системы класс Console позволяет взаимодействовать с этой консолью для ввода и вывода данных.
Когда вы запускаете программу в среде разработки или в командной строке, вывод программы будет отображаться в консоли операционной системы. Класс Console предоставляет удобные методы для чтения данных с консоли и записи данных на консоль.

Для консольного ввода-вывода этот класс использует уже запущенные в операционной системе Windows потоки System.in и System.out.
System.in представляет стандартный входной поток (стандартный ввод). Этот поток связан с консолью, и через него можно получать ввод с клавиатуры. Когда вы вводите данные с клавиатуры в командной строке Windows, эти данные поступают в System.in.

System.out представляет стандартный выходной поток (стандартный вывод). Этот поток используется для вывода данных на консоль. Когда вы выводите данные в командной строке Windows с помощью System.out.println() или других методов вывода, они отображаются в командной строке. 
import java.io.Console;

/**
 * Обратите внимание, что при запуске программы из среды разработки доступ к объекту Console может быть недоступен
 */
public class ConsoleExample {
    public static void main(String[] args) {
        /*объект получается при вызове System.console()*/
        Console console = System.console();
        if (console != null) {
            /**
             * Чтение строки с консоли
             * - readLine() - читает с консоли строку, которую ввёл пользователь. Сохраняет данные в String.*/

            String input = console.readLine("Введите значение : ");
            /**
             * printf - выводит на консоль указанные аргунен'l'Ы,
             используя формат, определяемый параметром*/
            console.printf("Вы ввели : %s\n", input);
        /*
        format() сохраняет отформатированную строку
         */
            console.format("Введите пароль: ");
        /*
        readPassword() - читает с консоли строку, которую ввёл пользователь.
         Отличие от метода readLine() в том, что вводимые пользователем символы
         не будут отображаться на консоли в целях безопасности.
         Сохраняет данные в char[] - массив байтов.
         */
            char[] password = console.readPassword();
            console.format("Вы ввели пароль: %s\n", new String(password));
            /**
             Возвращает ссылку на поток записи типа Wri ter,
             связанный с консолью
             */
            console.writer().println("Просто строка на консоли");
            // Очистка консоли
            console.flush();
        } else {
            System.out.println("Консоль не доступна.");
        }
    }
}

void
flush()
Сбрасывает консоль и заставляет немедленно записать любой буферизованный вывод.
Console
format(String fmt, Object... args)
Записывает отформатированную строку в выходной поток этой консоли, используя указанную строку формата и аргументы.

Удобный метод для записи отформатированной строки в поток вывода этой консоли с использованием указанной строки формата и аргументов.
