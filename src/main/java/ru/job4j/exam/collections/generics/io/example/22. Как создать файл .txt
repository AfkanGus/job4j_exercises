22 как создать файл
С помощью класса File можно работать с файлами и каталогами в файловой 
системе. Объект класса File - это абстрактное представление пути к файлам и каталогам, из этого объекта можно создать физический файл или каталог на диске. Объекты класса File неизменяемы - после создания файла или директории с указанным абстрактным путём, изменить этот путь будет невозможно. Также File не может изменять или получать содержимое файла, которое он представляет. То есть в этом классе описаны свойства самого файла, а не методы добавления, получения или редактирования его данных. С помощью объекта класса File можно получить доступ к сведениям о файле типа даты, времени, правах доступа, пути к каталогу и т.д.

Создать файл можно таким образом:

java
package ru.job4j.io;

import java.io.File;
import java.io.IOException;

public class FileExample {
    public static void main(String[] args) throws IOException {
        File file = new File("file.txt");
        file.createNewFile();
    }
}
В строке ниже в объект типа File инкапсулируется путь (в данном случае он состоит только из названия файла):
File file = new File("file.txt");
Теперь объект file хранит путь, который мы в него передали. Физически файл на диске еще не создан.
В следующей строке из объекта типа File создается файл на диске по пути, который хранит объект file:
file.createNewFile();
Чтобы создать файл не в корне проекта, нужно передать абсолютный или относительный путь до файла: 

java
File file = new File("src/main/java/ru/job4j/io/files/file.txt");


Абсолютный путь - путь, который указывает на одно и то же место в файловой системе, независимо от текущего каталога.

Например: C:\projects\job4j_exercises\src\main\java\ru\job4j\io\files\file.txt

Относительный путь - путь, указанный относительно текущего каталога.

Например: src\main\java\ru\job4j\io\files\file.txt
