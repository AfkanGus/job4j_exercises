7. Какие базовые методы существуют в Scanner


Класс Scanner в Java предоставляет ряд базовых методов для чтения различных типов данных из ввода. Вот некоторые из наиболее часто используемых методов:

next(): Читает и возвращает следующее слово (токен) из ввода, разделенное пробелами или другими разделителями.

nextInt(): Читает и возвращает следующее целое число из ввода.

nextDouble(): Читает и возвращает следующее число с плавающей точкой (double) из ввода.

nextLine(): Читает и возвращает следующую строку из ввода, включая пробелы.

hasNext(): Проверяет, есть ли еще токены во входных данных. Возвращает true, если следующий токен существует, и false в противном случае.

hasNextInt(), hasNextDouble(), hasNextLine(): Аналогично hasNext(), но проверяют наличие следующего токена определенного типа.
 // Устанавливаем регулярное выражение для считывания данных в формате "Имя-Возраст"
        scanner.useDelimiter("\\s*-\\s*");

skip(pattern): Пропускает ввод, который соответствует заданному регулярному выражению pattern.
  // Пропускаем пробел
        scanner.skip(" ");


useDelimiter(delimiter): Устанавливает новый разделитель токенов. По умолчанию разделителем является набор пробельных символов.

close(): Закрывает Scanner и освобождает связанные ресурсы.
