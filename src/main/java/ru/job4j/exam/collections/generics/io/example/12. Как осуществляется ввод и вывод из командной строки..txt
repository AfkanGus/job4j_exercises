12. Как осуществляется ввод и вывод из командной строки

Ввод и вывод из командной строки в Java осуществляются с помощью стандартных потоков ввода и вывода: System.in и System.out соответственно.
Ввод данных из командной строки осуществляется через стандартный поток ввода System.in. Этот поток представляет собой последовательность байтов, которые поступают в программу из командной строки. Для чтения данных из командной строки вы можете использовать классы Scanner или BufferedReader, чтобы считывать данные из System.in
Вывод данных на командную строку осуществляется через стандартный поток вывода System.out. Этот поток представляет собой последовательность байтов, которые отправляются из программы в командную строку. Для вывода данных на командную строку вы можете использовать методы System.out.print(), System.out.println(), System.out.printf()
