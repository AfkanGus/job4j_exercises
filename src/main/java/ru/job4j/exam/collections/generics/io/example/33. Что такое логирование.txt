33. Что такое логирование?

Логирование - это процесс записи в файл полезной информации о работе программы.
Полученный файл называют лог-файлом. Если приложение работает плохо, то первое что проверяют - это лог файл.
"Посмотри в лог" - это значит открыть файл логирования и посмотреть наличие Exception.

В Java есть удобная библиотека, которая позволяет решать эти вопросы.

Log4j - библиотека позволяет осуществить логирование процессов в приложении.



Запомните правило, если в проекте используется логгер, то для вывода ошибок или отладочной информации нужно использовать только логгер.

Придерживаетесь единого стиля во всем проекте.
